***Pygames is needed for this program.***

Welcome to our chess game! by Dakota Jones and Juan Luna.

***Simple Instructions***
1. Install Pygames.
2. Run Server.py located in Server Folder.
3. Run two instances of game.py located in Client Folder and hit start game.

***More Detailed Instructions***
To run this program you need to first run the server.py file in the Server folder. 
Then you can run the game.py file in the client folder.
To run the client.py, you will need to install the pygame library on your machine. 
You will need two instances running in order to start a game.
If you want to run the game on a different machine than your server go in the client.py file in the client folder and change the ip address.
