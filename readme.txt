Welcome to our chess game!

To run this program you need to first run the server.py file in the Server folder. Then you can run the game.py file in the client folder.
To run the client.py, you will need to install the pygame library on your machine. You will need two instances running in order to start a game.
If you want to run the game on a different machine than your server go in the client.py file in the client folder and change the ip address.
